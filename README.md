# ESP32 IoT Dashboard - Naan Mudhalvan Use Case 8

## 🚀 Overview
A professional-grade IoT implementation focusing on **Firmware Build Automation** and **Web-Based Hardware Control**.

## 🛠️ Features
* **CI/CD Pipeline:** Automated firmware building via GitHub Actions (`build.yml`).
* **Professional Environment:** Developed using PlatformIO IDE for robust library management.
* **Web Server UI:** Real-time GPIO control through a custom CSS/HTML "Dark Mode" dashboard.
* **Wi-Fi Connectivity:** Automatic network handshaking with status LED indicators.

## 📁 Technical Stack
* **Hardware:** ESP32 Dev Module
* **Framework:** Arduino / C++
* **Platform:** PlatformIO & VS Code
* **Automation:** GitHub Actions (Ubuntu-latest runner)
